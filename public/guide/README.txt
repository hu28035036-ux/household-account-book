가계부 작성 가이드 이미지 추가 방법
=========================================

이 폴더에 다음 9개 이미지 파일을 넣으면 /guide 페이지에 자동으로 보입니다.
파일이 없으면 회색 placeholder 가 표시됩니다 (앱은 정상 동작).

권장 사양
---------
- 형식: PNG 또는 JPG (PNG 권장, 투명 배경 가능)
- 가로 폭: 1200px ~ 1600px
- 비율: 16:9 또는 4:3
- 용량: 각 500KB 이하 (이미지 압축 도구 squoosh.app 등 사용)

파일명 (정확히 일치해야 함)
---------------------------
01-daily.png       매일 1번 짧게 적기 — 캘린더 입력 화면 캡처 또는 손글씨 일러스트
02-receipt.png     영수증 사진 → AI 업로드 흐름 화면
03-category.png    카테고리 선택 / 학습 규칙 적용 예시
04-budget.png      예산 설정 + 남은 예산 진행 바
05-recurring.png   고정 거래 등록 화면
06-small.png       작은 지출도 누적되면 큰 차이 — 그래프나 일러스트
07-import.png      CSV/XLSX 업로드 → 분석 후보로 들어오는 화면
08-candidates.png  분석 후보 검토 → 일괄 승인 화면
09-review.png      통계 페이지 회고 — 카테고리별 / 6개월 흐름

이미지 추가 후
--------------
파일을 이 폴더에 넣고 git push 하면 Vercel 배포 후 자동 표시됩니다.
새로고침 시 이미지가 보이지 않으면 브라우저 캐시 강제 새로고침 (Ctrl+F5).

캡처 팁
-------
- 화면 캡처는 모바일 사이즈(375px 폭)로 찍으면 가독성 좋음
- 민감정보(이름·실거래액)는 모자이크 처리
- 가짜 데이터로 캡처하는 게 가장 깨끗함
