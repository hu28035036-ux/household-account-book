가계부 작성 가이드 이미지 추가 방법
=========================================

이 폴더에 다음 9개 이미지 파일을 넣으면 /guide 페이지에 자동으로 보입니다.
파일이 없으면 회색 placeholder 가 표시됩니다 (앱은 정상 동작).

이 가이드는 "이 앱 사용법" 이 아니라 "가계부 자체의 보편 원칙" 입니다.
이미지도 종이 가계부·엑셀·다른 앱 모두에 통하는 일반 일러스트나 손글씨 예시가 좋습니다.

권장 사양
---------
- 형식: PNG 또는 JPG (PNG 권장, 투명 배경 가능)
- 가로 폭: 1200px ~ 1600px
- 비율: 16:9 또는 4:3
- 용량: 각 500KB 이하 (squoosh.app 등으로 압축 권장)

파일명 (정확히 일치해야 함)
---------------------------
01-purpose.png         왜 가계부를 쓰는지 — 목적 한 줄 적은 메모, 다이어리 표지 등
02-daily.png           매일 짧게 적는 습관 — 잠들기 전 1분 메모 / 손글씨 캘린더
03-income-expense.png  수입과 지출을 같은 곳에 — +/- 양쪽 컬럼 예시
04-category.png        분류(카테고리) 일관성 — 식비/교통/주거 분류 다이어그램
05-fixed-variable.png  고정 vs 변동 지출 — 좌우 비교 표 또는 차트
06-budget.png          월초 예산 한도 — 80만원 한도 메모, 진행 바
07-small.png           작은 지출 누적 — 영수증 더미, 1500원이 한 달이면 5만원 일러스트
08-payment.png         결제수단 구분 — 현금/체크/신용/이체 4가지 아이콘
09-review.png          월말 회고 — 가장 많이 쓴 카테고리 동그라미 친 메모

이미지 추가 후
--------------
파일을 이 폴더에 넣고 git push 하면 Vercel 배포 후 자동 표시됩니다.
새로고침 시 이미지가 보이지 않으면 브라우저 캐시 강제 새로고침 (Ctrl+F5).

이미지 소스 아이디어
--------------------
- 직접 손그림으로 그려서 스캔/촬영 (가장 따뜻한 톤)
- Canva / Figma 의 무료 일러스트 템플릿
- AI 이미지 생성 (DALL-E, Midjourney 등) — "minimal Korean household budget illustration"
- 종이 가계부 사진을 직접 찍어 모자이크 (실제감 ↑)

캡처/제작 팁
------------
- 민감정보(이름·실거래액)는 모자이크 또는 가짜 데이터로
- 9장의 톤(색감·선)을 통일하면 페이지 전체가 깔끔
- 각 이미지에 큰 제목 글자 1~2개 포함하면 캡션 안 읽어도 의미 전달
